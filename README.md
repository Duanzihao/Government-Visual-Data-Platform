# 政府大数据综合治理平台
该库为软件工程实验课政务系统项目使用代码  
## 运行环境
需 Python 3, django 2.2.6  

## 运行方式
在工程根目录下使用 Terminal 执行
`python manage.py runserver`  
然后即可使用浏览器访问 http://127.0.0.1:8000/
